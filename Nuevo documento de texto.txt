https://www.climate.gov/maps-data/dataset/sea-surface-temperature-and-sea-ice-coverage-map-viewer


https://www.ncdc.noaa.gov/data-access/satellite-data/satellite-data-access-datasets --> Este link contiene una cantidad de datos en formato NC, que al parecer se pueden abrir con python, y se pueden graficar.

https://psl.noaa.gov/psd2/coastal/satres/archive.html --> Este enlace contiene una cantidad considerable de imagenes relacionadas con diferentes tipos de variable (Vapor de agua integrado, Nubes de agua liquida, Precipitacion), esta disponible tanto para el oceano atlantico como para el oceano pacífico.

http://iridl.ldeo.columbia.edu/maproom/Global/Ocean_Temp/index.html --> Tiene un conjunto de datos que ellos mismos pintan en un mapa, pero los datos no son en formato de imagenes, sino en formato tabular, en algunas ocasiones dan una opción de descargar imagenes, pero termina siendo para abrir en programas tipo GIS

http://www.cgiar-csi.org/data/uea-cru-ts-v3-10-01-historic-climate-database#description --> Este dataset tiene dos variables adicionales a las que ya se coleccionaron (Presión de vapor y Frecuencia de humedad del día). Esta en formato tabular.

http://www.ntsg.umt.edu/freeze-thaw/visualizations/default.php --> En este link tienen en video la representación de un conjunto de datos tabular, se pueden obtener las imagenes a partir de este video?
